# Sea Surface Temperature Prediction

## Objective
Predict sea surface temperature using Machine Learning models.

## Models Used
- Linear Regression
- Decision Tree
- Random Forest
- KNN
- SVM
- AdaBoost
- XGBoost (Best Model)

## Techniques
- Data Cleaning
- EDA
- Outlier Removal (IQR)
- Feature Engineering
- Scaling
- Cross Validation

## Best Model
XGBoost achieved highest R² score.

## Deployment
Gradio GUI used for prediction.

## Dataset
Global Land Temperature Dataset (Kaggle)
